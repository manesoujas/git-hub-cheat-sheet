# 🚀 Ultimate Git & GitHub Cheat Sheet

A comprehensive, meticulously detailed single-page website that serves as a modern cheat sheet for Git and GitHub CLI (`gh`) commands. Built with a sleek slate color palette and responsive glassmorphism UI.

## ✨ Features
* **Zero Dependencies:** Written in pure HTML and CSS. No frameworks to install or compile.
* **Extensive Command List:** Covers everything from initial configuration and daily commits to advanced rebasing, stash management, and GitHub-specific terminal workflows.
* **Modern Aesthetic:** Features a frosted-glass UI (glassmorphism) over a deep dark-mode slate background, optimizing readability for developers.
* **Ready to Deploy:** Designed specifically to be hosted effortlessly on GitHub Pages.

## 🛠️ Commands Covered
This cheat sheet categorizes commands for immediate access:
1. **Setup & Init:** `config`, `init`, `clone`
2. **Stage & Snapshot:** `add`, `commit`, `status`
3. **Branch & Merge:** `branch`, `checkout`, `merge`, `rebase`
4. **Share & Update:** `remote`, `fetch`, `pull`, `push`
5. **Inspect & Compare:** `log`, `diff`, `show`
6. **Temporary Storage:** `stash`, `stash pop`, `stash list`
7. **Undoing Changes:** `reset`, `revert`, `clean`
8. **GitHub CLI (`gh`):** `auth`, `repo create`, `pr create`, `issue create`

## 🌐 How to Host on GitHub Pages

You can put this website live on the internet for free in under 2 minutes:

1. **Create a Repository:** Create a new public repository on your GitHub account (e.g., `git-cheat-sheet`).
2. **Upload the Files:** Upload the `index.html` and `README.md` files directly to the root of your new repository.
3. **Enable GitHub Pages:**
   * Go to your repository's **Settings** tab.
   * On the left sidebar, scroll down and click on **Pages**.
   * Under the **Build and deployment** section, look for **Source** and select `Deploy from a branch`.
   * Under **Branch**, select `main` (or `master`) and leave the folder as `/ (root)`.
   * Click **Save**.
4. **View Your Site:** After a few minutes, GitHub will provide you with a live URL (usually `https://[your-username].github.io/git-cheat-sheet/`).
