Ultimate Git & GitHub Hub

The exhaustive command reference for modern version control.

Setup & Init

git config --global user.name "[name]"
Sets the name attached to your commits.
git config --global user.email "[email]"
Sets the email attached to your commits.
git init [project-name]
Creates a new local repository.
git clone [url]
Downloads a project and its entire version history.

Stage & Snapshot

git status
Lists all new or modified files to be committed.
git add [file]
Snapshots the file in preparation for versioning.
git add .
Stages all current changes in the directory.
git commit -m "[message]"
Records file snapshots permanently in version history.
git commit -am "[message]"
Stages all modified files and commits them in one step.
git commit --amend
Modify the last commit (change message or add forgotten files).

Branch & Merge

git branch
Lists all local branches in the current repository.
git branch [branch-name]
Creates a new branch.
git checkout [branch-name]
Switches to the specified branch.
git checkout -b [branch-name]
Creates a new branch and switches to it immediately.
git merge [branch]
Combines the specified branch's history into the current branch.
git branch -d [branch-name]
Deletes the specified branch safely.
git rebase [branch]
Applies any commits of current branch ahead of specified one.

Share & Update (Remotes)

git remote add origin [url]
Connects your local repo to a remote server.
git remote -v
Shows the URLs of remote repositories.
git fetch
Downloads all history from the remote tracking branches.
git pull
Fetches and merges changes on the remote server to your working directory.
git push origin [branch]
Uploads all local branch commits to GitHub.
git push -u origin [branch]
Pushes and sets upstream tracking for future pulls.

Inspect & Compare

git log
Lists version history for the current branch.
git log --oneline --graph
Shows a compact, visual representation of the commit history.
git diff
Shows file differences not yet staged.
git diff --staged
Shows file differences between staging and the last file version.
git show [commit]
Outputs metadata and content changes of the specified commit.

Temporary Storage (Stash)

git stash
Temporarily stores all modified tracked files.
git stash pop
Restores the most recently stashed files and removes them from the stash list.
git stash list
Lists all stashed changesets.
git stash drop
Discards the most recently stashed changeset.

Undoing Changes

git reset [file]
Unstages the file, but preserves its contents.
git reset --hard [commit]
Discards all history and changes back to the specified commit. WARNING: Destructive.
git revert [commit]
Creates a new commit that undoes all of the changes made in a specific commit.
git clean -fd
Removes untracked files and directories from your working tree.

GitHub CLI (gh) Exclusives

gh auth login
Authenticate with your GitHub account.
gh repo create
Create a new GitHub repository from the command line.
gh repo clone [owner]/[repo]
Clone a repository to your local machine.
gh pr create
Create a pull request directly from the terminal.
gh pr list
View open pull requests for the repository.
gh issue create
Create a new issue on GitHub.
