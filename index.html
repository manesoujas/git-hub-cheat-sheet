Git & GitHub Architect Hub

The definitive tracking protocol reference with one-click copy capability.

Setup & Init

git config --global user.name "John Doe"
Sets the global name attached to your commits.
git config --global core.editor "code --wait"
Sets VS Code as the default editor for resolving merge conflicts.
git init
Initializes a new local repository in the current directory.

Stage & Snapshot

git add .
Stages all current changes in the directory.
git commit -m "Refactor servlet to use POJO-based architecture"
Records file snapshots permanently in version history.
git commit --amend --no-edit
Adds staged files to the very last commit without changing its message.

Branch & Merge

git checkout -b feature/flutter-ui-update
Creates a new feature branch and switches to it immediately.
git merge [branch-name] --no-ff
Merges a branch and forces a merge commit to preserve history context.
git branch -M main
Force renames the current branch to 'main'.

Rebase & Cherry-Pick

git rebase -i HEAD~3
Interactive rebase to squash, edit, or reorder the last 3 commits.
git rebase --onto main feature-old feature-new
Transplants a branch from an old base to a new base.
git cherry-pick [commit-hash]
Takes a specific commit from another branch and applies it here.

Stash & Worktrees

git stash push -m "WIP glassmorphism UI"
Temporarily stores modified files with a descriptive label.
git stash pop
Restores the most recent stash and removes it from the stash list.
git worktree add ../project-hotfix main
Checks out a branch in a separate physical folder (work on 2 branches at once).

Undo & Data Recovery

git reset --hard HEAD~1
Discards the last commit and all local uncommitted changes.
git revert [commit-hash]
Safely creates a new commit that undoes changes from a specific past commit.
git reflog
The safety net. Logs all HEAD movements to recover "deleted" commits or branches.

Debugging (Git Bisect)

git bisect start
Starts a binary search algorithm to find which commit introduced a bug.
git bisect bad
Marks the current commit as broken/containing the bug.
git bisect good [commit-hash]
Marks an older commit as working. Git will checkout a middle commit to test.

GitHub CLI (`gh`)

gh repo create [name] --public --clone
Creates a new public repo on GitHub and clones it locally instantly.
gh pr create -t "Add login UI" -b "Details"
Opens a Pull Request for your current branch directly from the terminal.
gh issue list --state open
Lists all currently open issues for the repository.
