Git & GitHub Architect Hub

The definitive tracking protocol reference. From basic versioning to advanced tree rewriting.

Setup & Init

git config --global user.name "[name]"
Sets the global name attached to your commits.
git config --global user.email "[email]"
Sets the global email attached to your commits.
git init
Initializes a new local repository in the current directory.
git clone [url]
Downloads a project and its entire version history.

Stage & Snapshot

git status
Lists all new or modified files to be committed.
git add [file]
Snapshots the file in preparation for versioning.
git add .
Stages all current changes in the directory.
git commit -m "[message]"
Records file snapshots permanently in version history.
git commit --amend -m "[new message]"
Modifies the last commit message or adds forgotten files to it.

Branch & Merge

git branch -a
Lists all local and remote branches.
git checkout -b [branch-name]
Creates a new branch and switches to it immediately.
git switch [branch-name]
Modern command to switch between branches.
git merge [branch]
Combines the specified branch's history into the current branch.
git branch -d [branch-name]
Deletes the specified branch safely (prevents deleting unmerged changes).

Advanced History Shaping

git rebase [branch]
Applies any commits of current branch ahead of specified one. Keeps history linear.
git rebase -i HEAD~[n]
Interactive rebase for the last n commits. Used to squash, edit, or reorder commits.
git cherry-pick [commit-hash]
Takes a specific commit from another branch and applies it to the current branch.

Share & Update (Remotes)

git remote add origin [url]
Connects your local repo to a remote server.
git fetch
Downloads history from remote tracking branches without merging them.
git pull --rebase
Fetches and merges changes, but puts your local commits on top of the remote ones.
git push -u origin [branch]
Uploads local branch commits and sets upstream tracking.

Inspect & Debug

git log --oneline --graph --all
Shows a highly readable, graphical tree view of all branches.
git diff [branch1]..[branch2]
Shows the content differences between two branches.
git blame [file]
Shows who changed what line in a file and in which commit.
git reflog
Logs all branch updates and HEAD movements. The ultimate undo safety net.

Stash Management

git stash push -m "message"
Temporarily stores modified files with a descriptive label.
git stash pop
Restores the most recently stashed files and removes them from the stash list.
git stash apply stash@{n}
Applies a specific stash from the list without deleting it.

Tags & Submodules

git tag -a v1.0.0 -m "Release 1.0.0"
Creates an annotated tag for a specific release point.
git push origin --tags
Pushes all local tags to the remote repository.
git submodule add [url]
Embeds another Git repository inside your current repository.
git submodule update --init --recursive
Initializes and clones submodules recursively.

Undoing & Cleaning

git reset [file]
Unstages the file, but preserves its contents in the working directory.
git reset --hard HEAD
Discards all local changes in tracked files and reverts to the last commit.
git revert [commit-hash]
Creates a new commit that safely undoes changes from a past commit.
git clean -fd
Force deletes untracked files and directories.

GitHub CLI (`gh`)

gh auth login
Authenticate with your GitHub account via terminal.
gh repo create [name] --public --clone
Creates a new public repo on GitHub and clones it locally instantly.
gh pr create --title "Title"
Create a pull request directly from the terminal.
gh pr checkout [pr-number]
Checks out a specific pull request branch locally to test it.
